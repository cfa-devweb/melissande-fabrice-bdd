<h2>MODULE BASE DE DONNEES</h2>
<p>Ce projet est dans le cadre d'une formation de Développement Web afin d'évaluer les compétences nécessaires sur la partie M5-S4 du Trello,<br>
"Mettre en place une base de données"</p>
####Modélisation du projet sous formes : MCD, MPD, MLD
<p>Mise en place des modèles de conception de schéma de base de données d'une bibliothèque "library"</p>
####Exportation du Modèle Logique de Données
<p>Une fois le MLD terminé, exporter en effectuant > File > Export > Forward Engineer SQL CREATE script..<br>
--> Nommer le fichier SQL exporté "library.sql"</p>
<h2>MODULE SCRIPT</h2>
<h3>Script de création de base de données</h3>
<p>Ce script est au format .batch et permet de lancer un script de création de base de données.<br>
--> Nommé "creation.bat"</p>
<h3>Script d'insertion des données au format .csv dans la table "books" de la base "library"</h3>
<p>Ce script est au format .batch et permet de lancer un script d'insertion de données dans une table.<br>
--> Nommé "insertion.bat"</p>
<h3>Script de sauvegarde de base de données</h3>
<p>Ce script est au format .batch et permet de lancer un script de sauvegarde de base de données.<br>
--> Nommé "save.bat"<br>
--> le fichier "sauvegarde.sql" sera créé automatiquement et rendu disponible à l'utilisateur.</p>
